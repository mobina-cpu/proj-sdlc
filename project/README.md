## 1- Clone project and install requirements:
```shell
git clone git@github.com:mobina-cpu/proj-sdlc.git
```
```shell
cd proj-sdlc
```
```shell
pip install -r requirements.txt
```
____

## 2- Run Project:
```shell
python download.py
```
#### if you want to add new url to download use this:
```shell
python new.py
```
____
### Project configs are exists in `.env`
